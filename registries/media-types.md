# MACP Media Type Registry

| Media Type | Description | Status |
|---|---|---|
| `application/macp+proto` | Canonical Protobuf-encoded MACP envelope or transcript | permanent |
| `application/macp+json` | Canonical JSON-mapped MACP envelope or transcript | permanent |
| `application/macp-agent-manifest+json` | JSON agent/runtime manifest document | provisional |
| `application/macp-mode-descriptor+json` | JSON mode descriptor document | provisional |
